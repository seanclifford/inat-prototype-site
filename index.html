Prototypes for iNaturalist tools

This is a set of tools that interface with iNaturalist to do some things the main site doesn't.
These have been created quickly, so don't expect a whole lot of polish.

Tools

  • Bulk send messages
  • Auto combine observations (Under construction)
  • Auto split observation (Idea - not implemented)

Configuration

  • Setup the iNaturalist network affiliation to use
  • Authenticate with your iNaturalist account to use some features
