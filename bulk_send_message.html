    • OR load from observations export file (JSON format) to message observers.
    • OR Enter comma separated list of observation ids to message observers.
    • OR Enter project id to load its members. This is in the url on the project page. e.g. stinkbugs-of-the-world
  3. Enter subject
  4. Enter message
